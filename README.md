# 🚀 AI-Powered Task Scheduler API

A production-ready ASP.NET Core Web API built with secure authentication and role-based authorization.

---

## 🔐 Features

- JWT Authentication
- Role-Based Authorization (Admin / User)
- Refresh Token System (Token Rotation)
- Secure Logout (Token Revocation)
- Clean Architecture Structure
- Entity Framework Core
- MySQL Database

---

## 🛠 Tech Stack

- ASP.NET Core Web API
- Entity Framework Core
- MySQL
- JWT Bearer Authentication
- Git & GitHub

---

## 🔑 Authentication Flow

1. User logs in → receives Access Token + Refresh Token
2. Access token expires (short-lived)
3. Client sends refresh token → gets new access token
4. Logout → refresh token revoked
